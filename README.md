# Expense Tracker Dashboard

This project was developed as part of my **internship**.

A simple and effective dashboard for tracking daily expenses and visualizing total spending.

## Tech Stack
- **Frontend:** React, Vite, CSS
- **Backend:** Node.js, Express
- **Database:** SQLite

## Features
- Add new expenses with descriptions and amounts
- Real-time calculation of total spending
- Persistent storage using SQLite
- Clean, minimal dashboard interface

## Setup
1. Backend: `cd backend && npm install && node index.js`
2. Frontend: `cd frontend && npm install && npm run dev`
